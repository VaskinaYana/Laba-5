# Задание 1

### Текст задачи

Написать программу вычисления суммы факториалов всех нечетных чисел от
1 до 9

### Алгоритм решения

1. Пользователь вводит число N с клавиатуры
2. Вызывается sum_fact_odd(N, Sum)
3. Если N = 1 → возвращается 1!
4. Иначе:
- вычисляется N!
- вызывается sum_fact_odd(N - 2)
- складываются результаты
6. Выводится итоговая сумма

# Задание 2

### Текст задачи

Определить количество раз, которое меняется знак в списке чисел, отличных от нуля. 

### Алгоритм решения

1.Пользователь вводит список чисел
2.Вызывается sign_changes(List, Count)
3.Берутся соседние элементы списка: X и Y
4.Проверяется:
- если X * Y < 0 → знак поменялся → +1
- иначе → без изменений
5.Рекурсивно обрабатывается оставшийся список
6.Когда остаётся 0 или 1 элемент → завершение
7.Выводится количество смен знака

# Задание 4

### Текст задачи

Витя, Юра и Миша сидели на скамейке. В каком порядке они сидели, если известно, что Юра сидел слева от Миши и справа от Вити?

### Алгоритм решения

1. Задать список: Витя, Юра, Миша.
2. Сгенерировать все возможные перестановки.
3. Для каждой перестановки:
- Найти позиции Вити, Юры и Миши.
- Проверить, что Юра правее Вити.
- Проверить, что Юра левее Миши.
4. Если условия выполняются — вывести порядок.
